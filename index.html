Nuestros proyectos

Chotuve Android

Aplicacion de Android desarrollada con React Native y Expo

Chotuve App Server

Servidor central encargado de la comunicacion con la aplicacion. Desarrollado con Flask y PostgreSQL

Chotuve Auth Server

Servidor encargado de la autenticacion de usuarios. Desarrollado con Flask y PostgreSQL

Chotuve Media Server

Servidor encargado del almacenamiento y administracion de los recursos Media. Desarrollado con NodeJS, Express y MongoDB

Chotuve Web Front

Front end administrativo, visualizacion de estadisticas y administracion de recursos. Desarrollado con React
